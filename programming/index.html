IMDb - Bash Script

This is a Bash script based on my one-line code to retrieve the sharpest image available from a series or movie IMDb-page. All you need to do is to type in the URL of the IMDB page and then the Script will find the URL of the sharpest image as well as the title of the series or movie and display it on the screen as well as saving it to a text file (.txt) by default, named output.txt.
